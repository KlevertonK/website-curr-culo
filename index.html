Conhecimentos

HTML 5

Utilizando do poder da nova versão do HTML, linguagem mais básica para organizar os blocos de informações do seu site. O esqueleto de qualquer site online atualmente, desde um site de grande porte como o da Hostinger Brasil, como o da RK Papel de Parede Líquido.

nível de conhecimento

CSS 3

Com essa linguagem de desenvolvimento front-end poderosa, fica cada vez mais simples e prático criar layouts simples, bonitos e intuitivos, aliando-se das possibilidades que o framework bootstrap possibilita sem contar a vasta gama de ícones disponíveis online.

nível de conhecimento

JavaScript

Com esta linguagem de programação fica muito mais fácil criar animações, funções que processam dados e apresentam a quem acessar seu site, se utilizar a biblioteca JQuery que ja trás muitas das funções prontas fica ainda mais rápido e fácil contruir elementos dinâmicos.

nível de conhecimento

Angular JS

Com este framwork baseado em JavaScript as possibilidades aumentam muito. Podendo-se criar elementos dinânmicos e estruturas mais complexas do que somente usando JavaScript. Permite o manejamento de dados de forma mais independente.

nível de conhecimento

PHP

Sendo a linguagem de programação voltada para comunicação com servidores mais utilizada da atualidade possibilita uma conexão mais segura ao banco de dados entre outras funções como envio de e-mails através de um formulário do site.

nível de conhecimento

Wordpress

Plataforma usada em grande parte dos casos por blogs e e-commerces, com uma interface simples e ideal para jornalistas e pessoas que desejam começar a por seu negócio ou idéias na web, com temas lindos e interativos e mil possibilidades.

formação e cursos

Centro universitário estácio de sá SC | 01/2013 á 01/2015

O Curso de Jogos Digitais me fez entender como a estrutura de programação funciona, como o navegador entende o que o desenvolvedor faz e mais do que isso me deu a base para desbravar este mundo de infinitas possibilidades que o desenvolvimento possibilita.

Criação e Desenvolvimento de sites

O curso engloba uma variedade de informação voltada para o desenvolvimento de sites dinâmicos e sistemas básicos com possibiliade de escalonamento, uso de tecnologias como AngularJS, pré-compiladores como sass e less fazem parte do conteúdo do curso, maestria com linguagens de desenvolvimento básicas como HTML5, CSS3 e JS, assim como frameworks como jQuery e d3.JS.
